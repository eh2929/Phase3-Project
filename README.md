# Job Tracker Application

This is a simple console-based job tracker application. It allows you to manage job applications and companies.

## Features

- Manage job applications: view, add, update, and delete job applications.
- Manage companies: view, add, update, and delete companies.

## Installation

1. Clone this repository.
2. Install the required dependencies by running `pip install -r requirements.txt`.

## Usage

1. Run the application by executing `python app.py`.
2. Follow the prompts in the console to manage job applications and companies.

## Dependencies

- Flask
- Flask-Migrate
- SQLAlchemy
- Rich
